# 🦈 THE MAGGI SAGA: Masala Pitch Game

Welcome to the **Maggi Cinematic Universe**! This is a chaotic, high-stakes interactive narrative game where you play a broke founder whose life-changing packet of Maggi has gone missing. 🍜🔥

![The Maggi Saga Intro](https://github.com/yadrikshauprety/magie/raw/main/src/assets/Screenshot%202026-04-24%20012819.png)

## 🎬 The Story
It's 2:14 AM. You're hungry. You're broke. You reach for your last yellow packet... and the drawer is **EMPTY**. 🌙

What follows is a multi-level odyssey of drama, legal battles, and billion-dollar pitches. Can you convince the Sharks to invest in your tragedy? Can you survive Amitabh Bachchan's interrogation? The fate of the economy (and your stomach) is in your hands.

## 🚀 Key Features
- **Cinematic Narrative**: A 4-level saga spanning from your bedroom to the Supreme Court.
- **AI-Powered Roasts**: Integrated with **Google Gemini AI** to roast your reactions in real-time. 🔥
- **Interactive Gameplay**: 
  - Fling **Nimita** off the screen when she says "I'm Out!" 🚀💨
  - Battle through **Level 3: KBC Maggi Edition**.
  - Navigate the **Supreme Court** for the ultimate verdict.
- **Premium UI/UX**: Built with a "Comic-Book" aesthetic, vibrant animations, and custom sound effects.
- **Responsive Design**: Playable on both desktop and mobile.

## 🛠️ Tech Stack
- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS + [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **AI Engine**: [Google Generative AI (Gemini)](https://aistudio.google.com/)
- **Voice**: Web Speech API for character narrations.

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yadrikshauprety/magie.git
   cd magie
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the Development Server**:
   ```bash
   npm run dev
   ```

## 🎮 How to Play
- **Level 1**: Discover the tragedy and choose your emotional reaction.
- **Level 2**: Pitch your "Missing Maggi" business to the Sharks. Don't let Ashneer Grover scare you!
- **Level 3**: Answer 5 high-stakes questions in KBC to fund your search.
- **Level 4**: Take the case to the Supreme Court for final justice.
